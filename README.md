Parametric Conic Curve Generator
Implementasi algoritma parametrik buat generate 4 kurva conic: lingkaran, elips, parabola, dan hiperbola. Dibuat buat tugas mata kuliah Computer Graphics, ngebandingin efek delta/step value ke kehalusan kurva dan beban komputasi.
Ada UI interaktif (HTML/CSS/JS + GSAP) buat ngatur parameter tiap kurva secara real-time, toggle radian/derajat, sama tabel koordinat hasil iterasi yang ke-update live pas slider digeser.
Stack: Vanilla JS, SVG, GSAP
