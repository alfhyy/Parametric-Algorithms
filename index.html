02 — Algoritma Parametric Kedua

Elips

Dua konstanta independen, a (radius horizontal) dan b (radius vertikal), membentuk kurva yang memanjang ke satu sumbu.

x = xc + a·cos(t)    y = yc + b·sin(t)

03 — Algoritma Parametric Ketiga

Parabola

Konstanta a menentukan jarak fokus dan seberapa lebar/sempit lengkungan parabola membuka.

x = a·t2    y = 2·a·t

Parameter Kurva

Jarak vertex ke fokus. a besar → parabola lebih lebar/landai.
t berjalan dari t1 ke t2. Tidak harus simetris terhadap 0.

Satuan Parameter

Parabola tidak memakai sudut; t di sini adalah parameter linear bebas, bukan radian/derajat.

04 — Algoritma Parametric Keempat

Hiperbola

Dua cabang terbuka menjauhi pusat. a mengontrol jarak puncak, b mengontrol kemiringan asimtot.

x = a / cos(t)    y = b·tan(t)

Parameter Kurva

Jarak pusat ke puncak. Mengontrol seberapa jauh kedua cabang terpisah.
Bersama a, menentukan kemiringan asimtot: y = ±(b/a)x.
t berjalan dari t1 ke t2. Dibatasi ±1.5 rad agar tidak menyentuh asimtot (singularitas cos t = 0 di ±π/2).
